## 새로운 커맨드: `/announce`
작업 기록(/save) 없이도 팀에 공지를 바로 보낼 수 있는 커맨드가 생겼습니다.

### 사용법
- `/announce` — 대화형으로 대상, 내용 입력
- `/announce engineering 에게: API 스펙 변경됐으니 확인해주세요` — 한 줄로 바로 전송
- 전체 팀(`all`) 또는 특정 역할에게 선택적으로 보낼 수 있습니다.

### 기존 /save와의 차이
- `/save` — 작업 기록 저장 + (선택) 공지
- `/announce` — 공지만 바로 작성

공지는 동일하게 `docs/shared/announcements/`에 저장되며, `/briefing`으로 확인 가능합니다.
