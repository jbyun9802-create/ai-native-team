## 변경된 것

1. **모든 .md 문서에 YAML 메타데이터가 필수입니다.**
   - 문서 최상단에 `---` 블록으로 date, type, role, topic 등을 넣어야 합니다.
   - 자세한 형식은 `docs/metadata-standard.md`를 참고해주세요.

2. **새 문서 type이 추가되었습니다.**
   - `research` — UX 리서치 산출물 (인터뷰 노트, 설문 결과 등)
   - `spec` — 구현 명세 (API 스펙, 인터랙션 스펙 등)

3. **미팅 원문 저장 규칙**
   - 미팅 끝나면 채팅 내보내기 / 트랜스크립트를 `docs/shared/meetings/날짜-주제/` 폴더에 txt로 넣어주세요.
   - 정리 안 해도 됩니다. 원문 그대로 붙여넣기만 하면 됩니다.
   - Fireflies 연동이 세팅되면 자동 저장됩니다.

## 왜?

나중에 AI한테 "이 기능 왜 이렇게 만들었어?" 같은 질문을 할 수 있게 하기 위한 기반 작업입니다. 메타데이터가 있어야 AI가 필요한 문서만 골라서 빠르게 답할 수 있습니다.
